# 📘 90 Days DevOps Learning Plan


## 🧠 My Understanding of DevOps and Cloud Engineering

### DevOps
DevOps is a culture and set of practices that combines development and operations to automate software delivery, improve collaboration, and ensure faster and reliable releases.

### Cloud Engineering
Cloud Engineering involves designing, deploying, and managing scalable applications using cloud platforms like AWS, Azure, or GCP.

- To gain high-demand skills in automation, cloud, and CI/CD  
- To build real-world projects and improve salary and career growth
